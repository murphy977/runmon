6类事件,挂了必达

完成、失败、报错输出、GPU 假死(进程活着却不干活)、日志静默、磁盘写满——任一发生,推到你手机。通道走 企业微信 / Bark / ntfy / Telegram,App 被系统杀了也收得到。

实时终端流 + 进度

手机上看终端输出实时滚动,进度条、loss、ETA 从日志自动解析,不用改一行训练代码。

资源曲线

GPU 利用率、显存、CPU、内存,实时曲线一眼看穿。

远程操作

停止、重跑、拉完整日志、跑完自动关机——租卡党省钱。

完整交互终端

手机上开一个真终端,跑任意命令,和 SSH 一样。

端到端加密 · 自托管

你的训练日志和命令全程加密,中转服务器只见密文。中转服务自己部署,数据不经过任何第三方。

零侵入接入

新任务 mon run -- python train.py 包一层就行;已经在 tmux 里跑了十小时的任务,mon attach 直接接管。

开始使用

三步:装 App、装 agent、扫码配对。

手机 App

安卓 · 25MB · 现代手机装 arm64 即可
下载 arm64 版
通用版 67MB arm32 老手机 x86 模拟器

服务器 agent

跑任务的机器上装
$ pip install runmon
$ mon pair
$ mon run -- python train.py
中转服务可自托管 · pip install runmon-relay
